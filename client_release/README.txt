vCampus 客户端部署说明 
 
1. 系统要求： 
   - Java 8 或更高版本 
   - Windows 操作系统 
 
2. 配置服务器连接： 
   - 编辑 config\config.local.properties 文件 
   - 修改 server.host 为服务器IP地址 
   - 修改 server.port 为服务器端口（默认8888） 
 
3. 启动客户端： 
   - 双击 start_client.bat 启动客户端 
 
4. 常见问题： 
   - 如果无法连接服务器，请检查网络连接和服务器地址配置 
   - 如果启动失败，请确保已安装Java运行环境 
 
5. 联系支持： 
   - 如有问题请联系系统管理员
